# タスク管理テスト

## 3状態チェックボックステスト

- [ ] 未完了タスク
- [-] 実行中タスク  
- [x] 完了タスク

## インデントテスト

  - [ ] ネストした未完了
  - [-] ネストした実行中
  - [x] ネストした完了

## 使い方

1. 任意のチェックボックス行にカーソルを置く
2. **Enterキー**を押して状態を循環させる：
   - `[ ]` → `[-]` → `[x]` → `[ ]` → ...

## その他のキーマップ

- `<leader>c` - チェックボックスの追加/削除
- `<leader>1~6` - ヘッダーレベル設定
- `<leader>0` - ヘッダー削除
- `<leader>*` - 箇条書き（*）
- `<leader>-` - 箇条書き（-）
