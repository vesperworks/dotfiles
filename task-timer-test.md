# タスクタイマー機能テスト

## 基本的な使い方

1. 任意のタスク行で **Enterキー** を押してチェックボックス状態を変更
2. `- [-]` 状態（進行中）になると自動でタイマーが開始される
3. 行末に `(45m)` のような経過時間が自動表示される
4. `- [x]` または `- [ ]` に変更するとタイマーが停止する

## テスト用タスク

- [-] レポート作成
- [ ] コードレビュー  
- [x] データ分析
- [ ] ミーティング準備

## デバッグ用コマンド

- `<leader>ta` - アクティブタイマー一覧表示
- `<leader>tq` - 全タイマー停止

## 使用方法

1. 上記のタスクの任意の行にカーソルを置く
2. **Enterキー**を押して `[ ]` → `[-]` に変更
3. 1分後に行末に経過時間が表示される
4. もう一度**Enterキー**を押して `[-]` → `[x]` に変更
5. 経過時間表示が消える

## 複数行テスト

複数行を選択(V)してからEnterキーを押すと、一括で状態変更可能：

- [-] タスク1
- [x] タスク2  
- [ ] タスク3
- [ ] タスク4

## 期待される動作
