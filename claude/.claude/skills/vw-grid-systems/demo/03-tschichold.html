The Grid as an Ethic

How nine grid systems shape the same content — a visual comparison

Thesis

The modular grid divides the type area into fields separated by gutters, creating a rational armature for text and image.

Argument
Order & philosophy

The modular grid divides the type area into fields separated by gutters, creating a rational armature for text and image. Each system — from Müller-Brockmann's Swiss precision to the medieval scribe's golden canon — encodes a different philosophy of order.

Today, CSS Grid and subgrid make every historical grid natively possible on the web. The question is no longer "can we?" but "which system fits the content?"
