# Worked Example: バッテリーパック $600/kWh → $80/kWh

## Overview

マスク本人が 2012 年の Kevin Rose インタビューで語った実例を、本スキルの 5 フェーズに当てはめた完全ワークスルー。**各 Phase の表をこの粒度で埋めるのが合格基準**。

背景: 2012 年当時、業界常識は「バッテリーパックは $600/kWh。過去もそうだったし将来も大きくは下がらない。だから EV は普及しない」だった。

## Phase 0: 適用判断

| 質問 | 判定 |
|------|------|
| 時間制約のある即時判断か？ | NO |
| 確立領域のルーティンか？ | NO（EV 普及という「新しいこと」をやろうとしている） |
| コスト構造を根本から変えたいか？ | **YES** |
| 「相場」が根拠の結論に違和感があるか？ | **YES**（「$600/kWh が相場」は誰かの実測ではなく業界の合意にすぎない） |

→ **続行**。理由: 「新しいこと（EV の大衆化）」であり、結論の根拠が相場＝類推である。

## Phase 1: 前提マッピング

| # | 前提 | 出所 | 検証方法 |
|---|------|------|----------|
| 1 | バッテリーパックは $600/kWh かかる | 類推（過去の購買実績・他社価格） | 構成素材まで分解して積算する |
| 2 | バッテリー価格は今後も大きく下がらない | 慣習（「ずっとこの水準だったから」） | 価格の構成要素のうち固定なのは何かを特定する |
| 3 | バッテリーはセルメーカーから完成品で買うもの | 慣習（業界の分業構造） | 内製・直接調達の場合のコストを試算する |
| 4 | リチウムイオン電池の素材構成はコバルト・ニッケル・アルミ・カーボン・ポリマー・鋼缶 | **事実**（化学組成） | — そのまま Phase 2 の部品になる |

5 つのなぜの例（前提 1）:
1. なぜ $600/kWh？ → サプライヤーの見積もりがそうだから
2. なぜ見積もりがその額？ → 市場価格がそうだから
3. なぜ市場価格がその額？ → 歴史的にそう推移してきたから
4. なぜ歴史的にそうか？ → **それ以上の理由がない** → 慣習/類推と確定

## Phase 2: 根本真実への分解

| カテゴリ | 事実 | 根拠 |
|----------|------|------|
| 数値的事実 | 構成素材は London Metal Exchange で調達可能 | LME 公開市場 |
| 数値的事実 | 素材スポット価格の積算 ≈ **$80/kWh** | コバルト・ニッケル・アルミ・カーボン・ポリマー・鋼缶の市場価格合計（2012 年時点） |
| 物理的制約 | セルのエネルギー密度はリチウムイオン化学の理論限界の範囲内 | 電気化学 |
| 論理的必然 | 完成品価格 ≧ 素材価格（製造が無料でも素材は買う必要がある） | 自明 |

**Idiot Index 計算**:

```
Idiot Index = $600 (完成品) ÷ $80 (素材) = 7.5 倍
```

→ 5 倍超。製造プロセス・流通・分業構造に大きなムダがある。**再構築の価値が高い**。

## Phase 3: ゼロからの再構築

目的の再確認: 「kWh あたりのパックコストを素材コスト下限に近づける」（手段＝バッテリーを安く買う、ではない）

| 案 | 構成 | 理論コスト | 矛盾 |
|----|------|-----------|------|
| A | 素材を取引所で直接調達し、セル工場を内製（後の Gigafactory） | $80 + 製造償却 → 中期 $150-200/kWh | なし |
| B | セル形状・パック構造を自社設計し歩留まりとパッケージ効率を改善 | 既存比 -30% | なし |
| C | 完成セルの長期大量契約で値引き | -10〜20% 止まり | **既存解の改善＝類推**（棄却） |

→ A + B を採用（実際の Tesla の戦略と一致）。

## Phase 4: 検証と反証

1. **反証条件**: 「素材→セルの製造に $520/kWh 分の本質的コスト（特許料・歩留まり物理限界等）があるなら誤り」→ 調査の結果、製造コストの大半はスケールと自動化で圧縮可能と判明
2. **未知の未知**: 規制（電池輸送規制）、コバルト供給の地政学リスク → リストに追加（後年、実際にコバルトフリー化が課題になった）
3. **抽象レベル**: $80/kWh は「明日の価格」ではなく「長期で漸近する下限」→ 結論を「中期 $150-200、長期 $100 以下」に補正

**結果（後日談）**: 2023 年の実勢パック価格は $128/kWh。1 周目の分析がほぼ的中した（確度: 高、の判定が正しかった例）。

## このワークスルーから学ぶ粒度の基準

- 前提は最低 3 件、うち 1 件以上が「事実」として Phase 2 に引き継がれる
- 数値は「$80/kWh」のように具体値で書く。「安くなるはず」は不合格
- Phase 3 で類推案（案 C）を**意図的に 1 つ書いて棄却する**と、ゼロベース案との違いが明確になる
- Phase 4 の補正（理論下限 → 実用予測）を必ず行う
