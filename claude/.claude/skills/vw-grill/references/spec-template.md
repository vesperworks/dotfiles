# Spec Template

## Overview

vw-grill の成果物 spec.md の固定セクション。保存先は `.brain/{project}/spec/{YYYY-MM-DD}-{slug}.md`。

## テンプレート

```markdown
---
date: {YYYY-MM-DD}
author: {モデル名}（vw-grill）
topic: {slug}
status: draft | approved
waves: {実施した wave 数}
---

# Spec: {タイトル}

## TL;DR（3行以内）

{何を作る/やるか、成功基準、次の一手}
```

## 記入ルール

- **根拠列は必須**。埋められない決定は「未決事項」に移す
- 却下した選択肢を残す（後続セッションが同じ検討を繰り返さないため）
- status は生成時 draft。ユーザーが承認したら approved に更新し、後続スキルへ渡す
