The Discipline

002

Modular Grid
+ Baseline

The modular grid divides the type area into fields separated by gutters, creating a rational armature for text and image. Müller-Brockmann insisted the grid was "an aid, not a guarantee" — it permits many solutions, but demands practice.

His system starts with the baseline: a fixed vertical rhythm (typically 8px on the web) to which every element snaps. Columns subdivide the horizontal axis; modules are the intersection of column and row.

The International Typographic Style — born in Basel and Zurich in the 1950s — stripped decoration to reveal structure. Grotesque sans-serifs (Helvetica, Univers, Akzidenz-Grotesk), flush-left ragged-right setting, and generous white space became its signatures.

Today, CSS Grid and subgrid make the modular grid natively possible on the web for the first time. The question is no longer "can we?" but "will we do it rigorously?"

Engineering the Grid

005

The Overlay
Problem

Most web "grid systems" draw guide lines as a full-viewport overlay disconnected from the content container. When the content sits in a centered max-width wrapper, the overlay columns drift at wider viewports.

The fix: put the overlay inside the same container, reading the same CSS variables. Then the overlay columns are the content columns at every width.

Optical
Alignment

A 180px headline whose layout box sits exactly on column line 1 still looks misaligned — because the letterform's ink is inset by its left side-bearing.

Runtime canvas measurement of actualBoundingBoxLeft corrects this, nudging the element so the ink, not the box, lands on the grid.
